# CPP2023-1-Equations

1. Console Application:
  `~$ <executable name> <arg1> <arg2>`

2. Behavior without command line arguments:
  - input &rarr; `std::cin`
  - output &rarr; `std::cout`

3. Behavior with one command line argument
  - input &rarr; `<arg1>`
  - output &rarr; `std::cout`

4. Behavior with two command line argument
  - input &rarr; `<arg1>`
  - output &rarr; `<arg2>`
